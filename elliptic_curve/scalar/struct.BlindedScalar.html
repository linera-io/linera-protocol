Expand description

Scalar blinded with a randomly generated masking value.

This provides a randomly blinded impl of Invert which is useful for e.g. ECDSA ephemeral (k) scalars.

It implements masked variable-time inversions using Steinโs algorithm, which may be helpful for performance on embedded platforms.
