Deserializes an IndexMap from an ordered sequence.

This function may be used in a field attribute for deriving Deserialize:

#[derive(Deserialize)]
struct Data {
    #[serde(deserialize_with = "indexmap::map::serde_seq::deserialize")]
    map: IndexMap<i32, u64>,
    // ...
}
