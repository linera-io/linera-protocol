Expand description

Serializes an IndexMap as an ordered sequence.

This function may be used in a field attribute for deriving Serialize:

#[derive(Serialize)]
struct Data {
    #[serde(serialize_with = "indexmap::map::serde_seq::serialize")]
    map: IndexMap<i32, u64>,
    // ...
}
