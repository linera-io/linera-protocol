pub fn last_error(code: c_int) -> Option<Error>

Returns the last error that happened with the code specified by code.

The code argument typically comes from the return value of a function call. This code will later be returned from the code function.

Historically this function returned Some or None based on the return value of git_error_last but nowadays it always returns Some so itโs safe to unwrap the return value. This API will change in the next major version.

pub fn code(&self) -> ErrorCode

Return the error code associated with this error.

An error code is intended to be programmatically actionable most of the time. For example the code GIT_EAGAIN indicates that an error could be fixed by trying again, while the code GIT_ERROR is more bland and doesnโt convey anything in particular.
