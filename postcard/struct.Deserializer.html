Struct Deserializer

Source 
pub struct Deserializer<'de, F: Flavor<'de>> { /* private fields */ }
Expand description

A serde compatible deserializer, generic over “Flavors” of deserializing plugins.

Please note that postcard messages are not self-describing and therefore incompatible with internally tagged enums.
