Expand description

Verify a message signature using the cert public key and the first TLS 1.3 compatible supported scheme.

This function verifies the dss signature over message using the subject public key from cert. Unlike verify_tls12_signature, this function only tries the first matching scheme. See WebPkiSupportedAlgorithms::mapping for more information.
