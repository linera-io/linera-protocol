Struct AlignedSerializer

Source 
pub struct AlignedSerializer<A> { /* private fields */ }
Expand description

A serializer made specifically to work with AlignedVec.

This serializer makes it easier for the compiler to perform emplacement optimizations and may give better performance than a basic WriteSerializer.
