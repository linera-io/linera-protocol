Enum BlockNumberOrTag

pub enum BlockNumberOrTag {
    Latest,
    Finalized,
    Safe,
    Earliest,
    Pending,
    Number(u64),
}
Expand description

A block Number (or tag - “latest”, “earliest”, “pending”)

This enum allows users to specify a block in a flexible manner.
