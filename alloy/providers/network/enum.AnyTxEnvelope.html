Enum AnyTxEnvelope

Source 
pub enum AnyTxEnvelope {
    Ethereum(EthereumTxEnvelope<TxEip4844Variant>),
    Unknown(UnknownTxEnvelope),
}
Expand description

Transaction envelope for a catch-all network.

Variants§

§

Ethereum(EthereumTxEnvelope<TxEip4844Variant>)

An Ethereum transaction.

§

Unknown(UnknownTxEnvelope)
