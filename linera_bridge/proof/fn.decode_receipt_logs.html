pub fn decode_receipt_logs(receipt_rlp: &[u8]) -> Result<Vec<ReceiptLog>>
Expand description

Decodes a receipt’s RLP and extracts its logs.

Handles EIP-2718 typed receipts (type byte prefix < 0x80).

This variant exists because when tests build MPT tries with multiple receipts (e.g. test_build_receipt_proof_multiple_receipts), each receipt needs a distinct cumulative_gas_used to produce different RLP encodings. Without that, all empty-log receipts would be byte-identical, making the trie degenerate.
