Enum TxKind

Source 
pub enum TxKind {
    Create,
    Call(Address),
}
Expand description

The to field of a transaction. Either a target address, or empty for a contract creation.

Variants§

§

Create

A transaction that creates a contract.

Call(Address)

A transaction that calls a contract or transfer.
