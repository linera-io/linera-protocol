Struct CopyToBytes

Source 
pub struct CopyToBytes<S> { /* private fields */ }
Expand description

A helper that wraps a Sink<Bytes> and converts it into a Sink<&'a [u8]> by copying each byte slice into an owned Bytes.

See the documentation for SinkWriter for an example.
