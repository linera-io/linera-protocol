# Conventions

Project conventions an agent must follow to produce mergeable PRs. These are distilled
**pointers** to the source of truth (`CONTRIBUTING.md`, `rustfmt.toml`, `clippy.toml`,
CI workflows) — when in doubt, that source wins.

| File | Scope |
|---|---|
| [`rust-style.md`](./rust-style.md) | Naming, formatting, API guidelines, commit/PR rules |
| [`metrics.md`](./metrics.md) | How to add a Prometheus metric |
| [`testing.md`](./testing.md) | How to write/run tests, CI gates |

Source of truth: `CONTRIBUTING.md`, `rustfmt.toml`, `clippy.toml`, `.github/workflows/`.
