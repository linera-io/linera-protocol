Struct SpannedValue

Source 
pub struct SpannedValue<T> { /* private fields */ }
Expand description

A wrapper for adding a Span to an arbitrary value.

Implementations are provided for all the parsing traits that simply delegate to T, capturing the inital Span from the ParseStream.
