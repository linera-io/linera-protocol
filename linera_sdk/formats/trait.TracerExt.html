Expand description

Extension methods on Tracer for tracing enums whose variant tags are not contiguous starting at zero.

The standard Tracer::trace_type discovers an enum’s variants by probing 0, 1, 2, … until each u32 index has been seen. With Keccak-derived stable tags those indices are not consecutive (they live in [2^27, 2^28)), so probing never terminates. This trait delegates to the enum’s StableEnumTrace impl, which drives tracing variant-by-variant via the enum’s Serialize impl.
