Asserts that an expression matches a given pattern.

Unlike assert_matches!, debug_assert_matches! statements are only enabled in non-optimized builds by default. An optimized build will omit all debug_assert_matches! statements unless -C debug-assertions is passed to the compiler.

See the macro assert_matches! documentation for more information.
