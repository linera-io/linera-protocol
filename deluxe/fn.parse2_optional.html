Parses a Rust type out of a token stream, returning a default value on failure.

Calls parse2 and returns the result. Upon failure, Default::default will be returned and any errors will be appended to errors. This function should be preferred when parsing multiple attributes, so the errors from all of them can be accumulated instead of returning early.
