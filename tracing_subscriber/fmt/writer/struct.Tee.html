Struct Tee

Source 
pub struct Tee<A, B> { /* private fields */ }
Expand description

Combines two types implementing MakeWriter (or std::io::Write) to produce a writer that writes to both MakeWriter’s returned writers.

This is returned by the MakeWriterExt::and method. See the method documentation for details.
