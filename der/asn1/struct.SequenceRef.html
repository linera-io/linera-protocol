Struct SequenceRef

Source 
pub struct SequenceRef<'a> { /* private fields */ }
Expand description

The SequenceRef type provides raw access to the octets which comprise a DER-encoded SEQUENCE.

This is a zero-copy reference type which borrows from the input data.
