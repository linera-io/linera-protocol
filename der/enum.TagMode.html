pub enum TagMode {

Expand description

Tagging modes: EXPLICIT versus IMPLICIT.

Variants§

§

Explicit

EXPLICIT tagging.

Tag is added in addition to the inner tag of the type.

§

Implicit

IMPLICIT tagging.

Tag replaces the existing tag of the inner type.
