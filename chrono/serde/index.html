Serialization/Deserialization with serde

The DateTime type has default implementations for (de)serializing to/from the RFC 3339 format. This module provides alternatives for serializing to timestamps.

The alternatives are for use with serde’s with annotation combined with the module name. Alternatively the individual serialize and deserialize functions in each module can be used with serde’s serialize_with and deserialize_with annotations.

Available on crate feature ‘serde’ only.
