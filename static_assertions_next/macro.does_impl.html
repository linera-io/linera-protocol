Macro does_impl 

Source 
macro_rules! does_impl {
    ($ty:ty: $($trait_expr:tt)+) => { ... };
}
Expand description

Returns true if the type does implement a logical trait expression.

§Examples

One can mimic assert_impl! using this macro:

const CONDITION: bool = does_impl!(u32: From<u8>);

const_assert!(CONDITION);
