Expand description

Same as assert_lt! in debug builds or release builds where the -C debug-assertions was provided to the compiler. For all other builds, vanishes without a trace.

Optionally may take an additional message to display on failure, which is formatted using standard format syntax.

ยงExample

#[macro_use]
extern crate more_asserts;

fn main() {
    // These are compiled to nothing if debug_assertions are off!
    debug_assert_lt!(3, 4);
    debug_assert_lt!(3, 4, "With a message");
    debug_assert_lt!(3, 4, "With a formatted message: {}", "oh no");
}
