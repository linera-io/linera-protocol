scylla::cluster

Module metadata

Source
Expand description

This module holds entities that represent the cluster metadata, which includes:

  • topology metadata:
    • Peer,
  • schema metadata:
    • Keyspace,
    • Strategy - replication strategy employed by a keyspace,
    • Table,
    • Column,
    • ColumnKind,
    • MaterializedView,
    • CQL types (re-exported from scylla-cql):
      • ColumnType,
      • NativeType,
      • UserDefinedType,
      • CollectionType,
