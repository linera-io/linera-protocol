Ascii
gRPC metadata values can be either ASCII strings or binary. Note that only visible ASCII characters (32-127) are permitted. This type should never be instantiated – in fact, it’s impossible to, because there’s no variants to instantiate. Instead, it’s just used as a type parameter for MetadataKey and MetadataValue.
