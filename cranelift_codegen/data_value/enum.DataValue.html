Enum DataValue

Source 
pub enum DataValue {
    I8(i8),
    I16(i16),
    I32(i32),
    I64(i64),
    I128(i128),
    F16(Ieee16),
    F32(Ieee32),
    F64(Ieee64),
    F128(Ieee128),
    V128([u8; 16]),
    V64([u8; 8]),
}
Expand description

Represent a data value. Where Value is an SSA reference, DataValue is the type + value that would be referred to by a Value.
