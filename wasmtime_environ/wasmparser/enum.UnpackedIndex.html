Enum UnpackedIndex

Source 
pub enum UnpackedIndex {
    Module(u32),
    RecGroup(u32),
    Id(CoreTypeId),
}
Expand description

The uncompressed form of a PackedIndex.

Can be used for match statements.

Variants§

§

Module(u32)

An index into a Wasm module’s types space.

§

RecGroup(u32)

An index into the containing recursion group’s elements.

§

Id(CoreTypeId)

A type identifier.
