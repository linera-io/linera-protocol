Enum CoreDumpValue

    Missing,
    I32(i32),
    I64(i64),
    F32(f32),
    F64(f64),
}
Expand description

Local and stack values are encoded using one byte for the type (similar to Wasm’s Number Types) followed by bytes representing the actual value See the tool-conventions repo for more details.
