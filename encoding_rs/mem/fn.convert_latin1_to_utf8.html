pub fn convert_latin1_to_utf8(src: &[u8], dst: &mut [u8]) -> usize
Expand description

Converts bytes whose unsigned value is interpreted as Unicode code point (i.e. U+0000 to U+00FF, inclusive) to UTF-8.

The length of the destination buffer must be at least the length of the source buffer times two.

Returns the number of bytes written.

Panics if the destination buffer is shorter than stated above.

Note that this function may write garbage beyond the number of bytes indicated by the return value, so using a &mut str interpreted as &mut [u8] as the destination is not safe. If you want to convert into a &mut str, use convert_utf16_to_str() instead of this function.
